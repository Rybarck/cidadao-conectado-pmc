Cidadão Conectado

Project Model Canvas - ODS 11: Cidades e Comunidades Sustentáveis

Grupo: Ricardo, João Pedro, Jader, Arthur Nardes

💬

JUSTIFICATIVAS
Passado

Muitas cidades enfrentam dificuldades na comunicação entre cidadãos e órgãos públicos para reportar problemas urbanos, como buracos, iluminação pública inadequada, coleta de lixo irregular e enchentes.

🎯

OBJ SMART

S: Permitir os cidadãos registrar, acompanhar e comunicar ocorreências urbanas, melhorando a comunicação com órgãos públicos.

📊

BENEFÍCIOS
Futuro

  • Maior agilidade na resolução
  • Transparência pública
  • Melhor qualidade de vida
  • Maior engajamento cívico
  • Redução de custos
📦

PRODUTO

Site interativo onde cidadãos podem registrar ocorrências urbanas com fotos e acompanhar o status em tempo real.

Funcionalidades:

  • Cadastro e login seguro
  • Formulário com geolocalização
  • Dashboard de acompanhamento
  • Notificações em tempo real
  • Estatísticas públicas
📋

REQUISITOS

Técnicos:

  • Hospedagem web estável
  • Banco de dados (SQL/NoSQL)
  • API de mapas
  • Framework web moderno
  • Protocolos de segurança
👥

STAKEHOLDERS
EXTERNOS
& Fatores externos

Usuários: Cidadãos que desejam reportar problemas urbanos

Stakeholders: Prefeituras, Secretarias Municipais, órgãos de infraestrutura urbana, associações comunitárias

👨‍💼

EQUIPE

Desenvolvedores: Ricardo e João Pedro

Responsabilidades: Frontend, Backend, Design, Testes
